ZTE SMS send Toolkit

1. Установить ZTEdrv.pkg - драйвер ZTE модема, позволяющий видеть в MacOS ZTE модемы в /dev/cu.ZTEUSBATPort_*
2. Выключить WiFi, с подключённым модемом перейти по ссылке http://192.168.0.1/goform/goform_process?goformId=MODE_SWITCH&switchCmd=FACTORY
Это переключит модем в FACTORY режим.
3. Подключить модем в Windows. Установить ZTEDrvSetup.exe - драйвер для ZTE модема в Windows.
4. Вставить в модем симку, которая не поддерживается. Включить dccrap.exe:
	Выбрать ZTE data...
	Нажать на лупу, чтобы нашло девайс
	Нажать Unlocking -> Unlock
5. После разблокировки модема снова перейти в MacOS. Теперь его можно использовать с любой SIM.
6. В скрипте sendsms собраны AT+ команды для того, чтобы послать смс с определённого девайса на определённый номер.

7. Для чтения SMS следует переключить память модема командой AT+CPMS=«SM»,»SM»,»SM»
